# 🗺️ Maps App – Custom Google Maps Integration

**Maps App** is a custom-built Android application that integrates Google Maps functionality to provide location-based services such as navigation, real-time location tracking, and map interaction. This project demonstrates the use of mapping APIs and location services in a modern mobile application.

---

## 📌 Overview

The **Maps App** leverages Google Maps APIs to deliver an interactive and user-friendly mapping experience. It allows users to explore locations, view maps, and interact with geographic data in real time.

This project is designed for educational purposes and showcases practical implementation of location-based features in Android development.

---

## ✨ Features

### 📍 Real-Time Location

* Display user's current location on the map
* Continuous location updates

### 🗺️ Interactive Map

* Zoom in/out and navigate freely
* Switch between map types (Normal, Satellite, Hybrid)

### 📌 Markers

* Add and display markers on specific locations
* Show location details with info windows

### 🔎 Location Search

* Search for places and addresses
* Move camera to selected location

### 🌐 Map Controls

* Compass and location button
* Gesture controls for smooth interaction

---

## ⚙️ Tech Stack

* **Platform:** Android (Java)
* **IDE:** Android Studio
* **APIs Used:**

  * Google Maps SDK for Android
  * Google Places API *(optional)*
  * Location Services (FusedLocationProviderClient)
